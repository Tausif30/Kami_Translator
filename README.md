## How to Use

1. **Translate a page**: Click the extension icon, select target language, and click "Translate Page"
2. **Auto-scroll translation**: After translating, scroll down 50% of viewport height to automatically translate newly visible content
3. **Restore original**: Click "Restore Original" button to revert translations
4. **Translate selection**: Right-click selected text â†’ "Translate [text]"
